## ğŸš€ How to Run

This project uses **Bun** for package management and as the runtime.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ahmad-006/NodeJS-mastery-journey.git
    cd NodeJs-Mastery
    ```

2.  **Install dependencies:**
    ```bash
    bun install
    ```

3.  **Start the server:**
    To run the latest module (09-Dynamic_Routes) with watch mode:
    ```bash
    bun run start
    ```

4.  **Visit the app:**
    Open your browser and go to `http://localhost:3000`
